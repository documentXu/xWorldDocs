### `plot`标绘点操作方法


>`opens` 打开多个标绘点
```javascript
plot.opens([service]);
```
>`open` 打开单个标绘点
```javascript
plot.open(service);
```

>`close` 关闭单个标绘点
```javascript
plot.close(service);
```
>`jump` 标绘点跳转
```javascript
plot.getMap(service);
```
