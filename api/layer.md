### `layer`图层操作方法


>`opens` 打开多个图层
```javascript
layer.opens([service]);
```
>`open` 打开单个图层
```javascript
layer.open(service);
```

>`close` 关闭单个图层
```javascript
layer.close(service);
```
>`getMap` 生成地图服务
```javascript
layer.getMap(service);
```
